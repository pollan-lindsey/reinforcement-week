• What is a function in JS?

    • A function is a statement/set of statements that perform a task which can be referred to at any time, which hass both an input and output value.


  • In your own words, why do we use functions in Javascript?

    • We use functions so we can repeat an action or task without having to write that action or task more than once. It saves time and space when writing code.


  • In your own words, when should we use a function?

    • A function should be called whenever we want to perform an action or task.


  • What is a higher order function in Javascript?

    • A higher order function is a function which uses a function as an arugument and/or returns a funtion.
